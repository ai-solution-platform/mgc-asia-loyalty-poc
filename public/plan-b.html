04 — Infrastructure

On-Premise Architecture

Deployed on MGC-ASIA infrastructure — Full data sovereignty, zero cloud dependency

Benefits of On-Premise

  • ✓ Data sovereignty — all data stays within MGC-ASIA network
  • ✓ Full compliance with Thai PDPA requirements
  • ✓ No recurring cloud costs (AWS/Azure/GCP)
  • ✓ Full control over infrastructure and security
  • ✓ No vendor lock-in on infrastructure layer

Infrastructure Requirements

  • • Application Server: 4 vCPU, 16GB RAM, 100GB SSD (x2 for HA)
  • • Database Server: 4 vCPU, 32GB RAM, 500GB SSD
  • • Redis Cache: 2 vCPU, 8GB RAM
  • • Load Balancer + SSL Certificate
  • • Network: Internal LAN access + public endpoint for mobile app API
